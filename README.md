# Chat bot

Chat bot with RASA v-3.0.4

To active conda

```bash
conda activate rasatest
```

here "rasatest" is your virtual env name

To deactive conda

```bash
conda deactivate

```

To run rasa

```bash
rasa shell
```

To train model

```bash
rasa train
```

To run and enable rasa API

```bash
rasa run --cors "*" --enable-api
```
